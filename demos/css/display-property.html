CSS Display Property: block vs inline vs inline-block

display: block

Three block elements:
Box 1
Box 2
Box 3
Key Characteristics:
  • Takes up full width available
  • Starts on a new line
  • Width and height can be set
  • Respects all margins and padding

display: inline

Three inline elements:
Box 1 Box 2 Box 3
Key Characteristics:
  • Only takes width of content
  • Flows inline with text
  • Width and height are IGNORED
  • Vertical margins don't work
  • Cannot force line breaks

display: inline-block

Three inline-block elements:
Box 1
Box 2
Box 3
Key Characteristics:
  • Flows inline like text
  • Respects width and height
  • Respects all margins and padding
  • Best of both worlds!
