# 01 - Introduction

## Tag vs Element

- **Tag**: The markup notation used to define an element. Tags use angle brackets (e.g., `<h1>`, `</h1>`)
  - Opening tag: `<h1>`
  - Closing tag: `</h1>`
  - Self-closing tag: `<br />`
- **Element**: The complete structure including opening tag, content, and closing tag (e.g., `<h1>My Name</h1>`)

## Concepts Covered

This folder contains exercises demonstrating basic HTML concepts:

- **HTML Document Structure**: `<!DOCTYPE>`, `<html>`, `<head>`, and `<body>` tags
- **Meta Tags**: Character encoding (`<meta charset>`) and viewport settings
- **Page Title**: Using the `<title>` tag
- **Headings**: `<h1>` (header 1) elements for structure and hierarchy
- **Anchor Tag (`<a>`)**: Used to create hyperlinks
- **Attributes**: Name-value pairs that provide additional information to elements
  - `href` (hypertext reference): Used in anchor tags to specify the link destination
  - `src`: Used in image tags to specify the image source
  - `alt`: Provides alternative text for images
- **Line Breaks**: `<br />` tag for breaking to a new line
- **Images**: `<img>` tag for embedding images
- **Email Links**: Using `mailto:` in href attributes
- **Lists**:
  - `<ul>`: Unordered list
  - `<ol>`: Ordered list
  - `<li>`: List item tag
